# le_bdf
## Simple English
manipulate BDF class in Liberty Eiffel

## 日本語での説明
BDFファイルをビットマップにするLiberty Eiffelのクラス(bdf.e)です。
説明は面倒になったので、サンプルのbdf_test.eを見てください。
